## Правило
Wildberries `s2s-api-auth-stat` (news #281) заблокировал IP Replit (35.234.214.215, GCP Europe-West1).
Node.js запросы к WB API с сервера падают с 403.

**Why:** WB ввёл блокировку GCP-диапазонов для предотвращения автоматизированного доступа.

**How to apply:**
- `local-wb-proxy.mjs` + `start-wb-proxy.bat` в корне проекта — запускает прокси на localhost:3001 у пользователя
- `wb-api.ts` на фронтенде сначала пингует `/api/wb/health` (через локальный прокси), при успехе использует localhost:3001, иначе fallback на серверный прокси
- UI показывает 🟢/🔴 статус прокси с обновлением каждые 5 сек

## Что нужно сделать при продолжении
- Проверить, запускает ли пользователь `start-wb-proxy.bat` перед загрузкой WB
- Если локальный прокси не запущен, WB данные недоступны (нет обходного пути с сервера)
